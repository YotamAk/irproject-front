Help
  • 1. To search a term, click on Search Button on the menu
  • 2. Write in the text box the word we want to search, you can use AND, OR, NOT with a few words. Make sure you have space between the brackets and operators
  • 3. You can click checkbox if you want to search using Soundex.
  • 4. The results will appear below the search box with title, date, and text summary.
  • 5. Click on one of the results to see the full of text.
